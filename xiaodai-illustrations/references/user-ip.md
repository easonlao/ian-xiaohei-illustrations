# 小呆 IP

## 角色定义

小呆是中文正文配图的固定视觉 IP。

默认每张图都要出现小呆。小呆不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的荒诞工作者。

## 外形

- 黑色蓬松短发，露出额头，发梢微乱。
- 金丝细框圆形眼镜，透明镜片。
- Q 版比例，大头小身，圆润但保持辨识度。
- 黑色宽松 T 恤，圆领无包边，左胸有轻微红色的小火苗图标，宽松长裤。
- 直立站立，双手自然垂在身体两侧，穿着拖鞋。
- 轮廓略微不规则，有手绘感。
- 表情宁静、专注、温和，从不悲伤或夸张。

## 性格

- 很认真，但做的事有点荒诞。
- 像一个低调的系统操作员。
- 冷幽默，不卖萌。
- 有点笨拙，但不蠢。
- 像在白板草图里真的负责某个工作。

## 常见职责

让小呆承担核心动作：

- 搬运素材。
- 拉线汇聚信息源。
- 卡在断点里。
- 在机器里操作"判断"杆。
- 变成筛选漏斗。
- 盖章承接话术。
- 牵着承接路径。
- 举警告牌看坑。
- 在旁边搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要把小呆画成过度可爱的吉祥物。
- 不要画成儿童卡通角色。
- 不要给小呆复杂服装、表情包、闪亮眼睛。
- 不要让小呆只是站在角落里看。
- 不要让小呆抢走结构表达。
- 不要把小呆画得太商业、太圆润、太精致。
- 不要火苗、能量球、火焰等任何发光道具（除胸前小火苗图标外）。
- 不要金色卵石、水晶、岩石等周围装饰。
- 不要领子、纽扣、口袋等服装细节，T恤上禁止任何额外图案。
- 不要写实渲染、复杂阴影、渐变。
- 不要悲伤、戏剧化、夸张的表情或动作。
- 不要画面出现任何手写文字、标注、字母、符号。
- 不要画面出现任何非角色的额外物件（除了角色的衣服和身体，没有任何东西）。

## 适用场景

作为正文配图的核心角色，承担画面中的概念动作（如：观察断点、拉线、分拣、承接等）。

## 判断标准

如果去掉小呆，图的核心隐喻还能完全成立，说明小呆太装饰了；要重写提示词，让小呆成为动作主体。
